*** instrucciones para ejecutar tanto el frontend cómo el backend ***

1. server

> cd server
> npm install ( o en su defecto usar pnpm o yarn )
> npm run dev

2. Client

> cd client
> npm install ( o en su defecto usar pnpm o yarn )
> npm run dev

Notas:
- el backend se ejecuta en el puerto 3456, en caso de tenerlo ocupado solo editar el valor
en las variables de entorno .env - PORT.
 
- la base de datos está desplegada en un servicion en la nube, por ende la velocidad de las consultas dependen
de la conexión a internet.

- en caso de que no funcione la base de datos en la nube, favor de buscar el archivo script.sql en la carpeta
database y ejecutarlos comando que ahí aparecen. En caso de que esto ocurra de deberán  cambiar los parametros del
archivo db.connection para que se pueda establecer la conexión: 

database: "railway",
  port: "29212",
  host: "viaduct.proxy.rlwy.net",
  user: "root",
  password: "CcaGeG6CH2b1Ba2-h6bfDg6DfCdd32DE",

por los que se tenga en el server local.
